Butter Cli
============

关于本项目(About)
---------
Butter Cli是一个使用模板初始化node项目的命令行工具
Butter cli is a cli(Command-line interface) tool for initializing node project with template

安装(Install)
---------
```
npm install butter-cli -g
```

使用(How to use)
---------
```
butter-cli [project-name]
```

获取帮助(Get help)
---------
```
butter-cli
```

测试(Test)
---------
## 单元测试(Unit test)
```
npm test
```
## 覆盖率测试(Unit test with coverage)
```
npm run test-cov
```
